========================================
INSTALLATION STEPS (Windows)
========================================

METHOD 1: AUTOMATIC SETUP (Recommended)
----------------------------------------

1. Copy the entire project folder to your new laptop

2. Open Command Prompt (cmd) in the project folder
   - Right-click on folder ‚Üí "Open in Terminal"
   - Or: Win+R ‚Üí type "cmd" ‚Üí navigate to folder

3. Run the setup script:
   setup.bat

4. Wait for installation (5-10 minutes)
   - It will download ~550MB of packages
   - Creates virtual environment
   - Installs all dependencies

5. When prompted:
   - Install Google Vision? ‚Üí Type 'n' (unless you need it)
   - Install Gemini AI? ‚Üí Type 'y' (for answer evaluation)

6. Done! ‚úì


METHOD 2: MANUAL SETUP
----------------------------------------

1. Open Command Prompt in project folder

2. Create virtual environment:
   python -m venv venv

3. Activate virtual environment:
   venv\Scripts\activate

4. Upgrade pip:
   python -m pip install --upgrade pip

5. Install all requirements:
   pip install -r requirements.txt

6. Download TextBlob data:
   python -m textblob.download_corpora

7. Install Gemini for AI evaluation:
   pip install google-generativeai

8. Done! ‚úì


========================================
SETUP API KEY (For AI Evaluation)
========================================

1. Get FREE API key:
   Visit: https://aistudio.google.com/app/apikey
   - Sign in with Google account
   - Click "Create API Key"
   - Copy your key

2. Set the API key:
   
   OPTION A - Temporary (for current session):
   set GEMINI_API_KEY=your_api_key_here
   
   OPTION B - Permanent (recommended):
   - Press Win+X ‚Üí System
   - Advanced system settings
   - Environment Variables
   - New User Variable:
     Name: GEMINI_API_KEY
     Value: your_api_key_here
   - Click OK
   - Restart Command Prompt

========================================
TROUBLESHOOTING
========================================

PROBLEM: "Python is not recognized"
SOLUTION: 
- Reinstall Python with "Add to PATH" checked
- Or add Python to PATH manually

PROBLEM: "pip is not recognized"  
SOLUTION:
python -m pip install --upgrade pip

PROBLEM: "Module not found"
SOLUTION:
venv\Scripts\activate
pip install -r requirements.txt

PROBLEM: "Port 8000 already in use"
SOLUTION:
- Close other applications using port 8000
- Or edit config.py and change PORT = 8001

PROBLEM: Gemini API not working
SOLUTION:
- Check API key is set: echo %GEMINI_API_KEY%
- Verify key at: https://aistudio.google.com/app/apikey
- Make sure to use run_with_ai.bat

PROBLEM: Slow installation
SOLUTION:
- Normal! EasyOCR downloads ~400MB of models
- Wait 5-10 minutes for first installation
- Subsequent runs will be fast

========================================
LINUX/MACOS INSTALLATION
========================================

1. Make scripts executable:
   chmod +x setup.sh run.sh

2. Run setup:
   ./setup.sh

3. Set API key:
   export GEMINI_API_KEY=your_api_key_here

4. Run application:
   ./run.sh

5. Open: http://localhost:8000


========================================
OFFLINE INSTALLATION (No Internet)
========================================

If the new laptop has no internet:

1. On a laptop WITH internet:
   pip download -r requirements.txt -d packages/

2. Copy entire project + packages folder to new laptop

3. On new laptop:
   pip install --no-index --find-links=packages/ -r requirements.txt
